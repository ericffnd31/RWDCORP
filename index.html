⚠️ Setup Belum Selesai (Khusus Admin)

Halaman ini ditujukan untuk staff, tapi URL backend (Google Apps Script) belum ditanam di dalam kode. Staff tidak akan bisa login atau absen sebelum ini diperbaiki.

Cara perbaiki (sekali saja, oleh admin):
1. Buka file index.html di editor teks.
2. Cari baris: const API_URL = "GANTI_DENGAN...";
3. Ganti dengan URL Web App dari Apps Script Anda.
4. Simpan file, lalu re-upload / re-deploy ke Vercel.
